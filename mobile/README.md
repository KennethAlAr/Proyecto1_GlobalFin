# App Móvil B2C

Carpeta para el cliente móvil, usando React Native o Flutter.

## Estructura propuesta

- `app/` o `lib/`: código fuente de la aplicación.
- `assets/`: imágenes y recursos.

Pasos iniciales:
1. Inicializar proyecto con `npx react-native init` o `flutter create`.
2. Implementar primera pantalla (login/dashboard) con estilos corporativos.

> Se recomienda crear un prototipo ligero con hot-reload para iterar rápidamente.
