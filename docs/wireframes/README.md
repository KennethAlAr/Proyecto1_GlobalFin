# Wireframes Low‑Fi

Esta carpeta contiene wireframes en formato SVG para arrancar la validación de flujos de usuario.

Archivos:

- `login.svg` — pantalla de inicio de sesión minimalista.
- `dashboard.svg` — vista principal del dashboard B2B con tabla y panel lateral.
- `transactions.svg` — vista de tabla de transacciones con filtros y detalles.

Guía rápida:

1. Revisar estos wireframes con los stakeholders y anotar cambios.
2. Transferirlos a Figma (web) para iterar Hi‑Fi. Puedes usar los SVG como base importándolos a Figma.
3. Después de validar, prepararemos componentes interactivos y flujo de datos para el prototipo.
