# Dashboard B2B - Escritorio

Carpeta dedicada al cliente de escritorio (JavaFX o .NET MAUI según evaluación).

## Estructura propuesta

- `src/`: código fuente del cliente.
- `resources/`: FXML, estilos, assets.

Los siguientes pasos:
1. Configurar un proyecto Java/JavaFX con Maven o Gradle.
2. Añadir un `Main.java` con una ventana vacía para GlobalFin.

> Nota: el equipo puede elegir .NET MAUI si prefiere C#/.NET en lugar de Java.
