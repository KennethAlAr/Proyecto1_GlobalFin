# Proyecto1_GlobalFin

Este repositorio contiene el material inicial para el *Sprint Inicial* de
modernización de las interfaces de GlobalFin en GlobalFin Bank.

## Contenido

- `docs/` – estrategia del sprint, decisiones tecnológicas y notas de
	consultoría. Wireframes y diagramas.
- `desktop/` – cliente de escritorio (JavaFX/.NET MAUI). Imágenes hi-fi de la aplicación de escritorio.
- `mobile/` – cliente móvil (React Native/Flutter). Imágenes hi-fi de la aplicación móvil.
- `Respuestas GlobalFin.md`- Respuestas a las preguntas del CEO y justificación de las tecnologías utilizadas.

## Primeros pasos

1. Revisar `docs/SPRINT.md` para entender la planificación UX y técnica.
2. Generar wireframes y prototipos con Figma.
3. Configurar el entorno de desarrollo con IntelliJ IDEA (ver docs).
